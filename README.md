# dd-test-high-light

一个包含 Spring Framework 消息传递接口示例和自定义 404 错误页面的测试项目。

## 📋 项目内容

这个仓库包含以下文件：

### Java 文件
- **Test.java** - Spring Framework 的 `MessageChannel` 接口实现示例
  - 定义了消息发送的方法
  - 包含超时控制机制
  - 基于 Apache License 2.0

### HTML 文件  
- **404.html** - 自定义的 404 错误页面
  - 响应式设计，适配各种设备
  - 简洁友好的错误提示

## 🛠️ 技术栈

- **Java** (63%) - 主要编程语言
- **HTML** (37%) - Web 页面
